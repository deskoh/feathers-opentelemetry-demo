# FeathersJS OpenTelemetry Demo

> Works for Feathers v4 (Crow)

## Quickstart

```sh
# Install dependencies
npm ci

# Start Zipkin
docker run -it --name zipkin -p 9411:9411 openzipkin/zipkin:2.23

# Start server
npm run dev

# Curl server
curl http://localhost:3030/messages

# View trace @ http://localhost:9411/
```
